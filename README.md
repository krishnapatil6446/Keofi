### Run Locally

To run **Keofi** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/keofi.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/keofi.git
```
